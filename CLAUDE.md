

# 项目简介

`skill-box` 是基于 Wails v3 + Gin + Vue 3 + Pinia 的桌面/Web 双形态应用,
本仓库为 Go 1.25 工作区,业务代码全部在 `api-server/` 与 `frontend/`。

wails3 文档地址：https://v3.wails.io/quick-start/why-wails/
