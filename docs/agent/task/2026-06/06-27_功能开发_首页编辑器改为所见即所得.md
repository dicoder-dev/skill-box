# 首页技能编辑改为 Tiptap 所见即所得编辑器

**日期:** 2026-06-27
**状态:** 进行中

## 1. 需求

参考 `/Volumes/MyDrive/Home/dicoder/projects/opcbase` 项目的 `ArticleEditor.vue`(基于 Tiptap 实现),
把首页 SkillsView 的"内联编辑 body"从现在的纯 textarea 升级为 Tiptap 所见即所得编辑器。

要求:
- 保持现有 description / triggers 文本编辑框不动(已经做过的体验调整)
- 只把 `body`(SKILL.md 正文部分)的编辑从 textarea 换成 Tiptap WYSIWYG
- 编辑器要支持:标题(H1-H3)、加粗/斜体/删除线、有序/无序列表、引用、代码块/行内代码、链接、图片
- 工具栏 UI 参考 opcbase 的实现,但视觉风格要符合 skill-box 的简洁风格
- 保持与 skill-box 现有的 skills/前端技术栈一致(Vue 3 + Vite,不使用 emoji)
