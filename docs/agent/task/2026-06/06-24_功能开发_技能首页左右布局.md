# 技能首页改左右布局

**日期:** 2026-06-24
**状态:** 已完成

## 1. 需求

用户希望把技能首页(SkillsView)从"表格 + 弹窗"模式改成左右布局:

- **左侧:**技能列表;顶部"新建技能 / 导入技能"按钮;中部搜索框;下方列表项(名称 + 版本 + 源/作用域徽章,选中态高亮)
- **右侧:**选中后:
  - 顶部 toolbar:技能名 + 版本信息;右侧操作图标(测试 / 打标签 / 在文件夹打开 / 删除,hover 显示文字)
  - scope chips(全局 + 1...N 项目,多选)
  - 标签列表
  - 下方渲染 SKILL.md 的 body 内容(不含 frontmatter),简单 markdown
- **风格:**黑灰白,圆角 6px,细边框,与现有 style.css 兼容;无 emoji;iconify mdi 系列

## 4. 问题与方案

**问题 1:listSkills 后端不接受 scope 参数。** 控制器 cskill/list_skills.a.go 只看 keyword + page + size,直接拉全表(扫描文件)。前端不再传 scope,把 size 提到 200 拉全,前端本地按关键字二次过滤。

**问题 2:platform/index.js 导出 named,不是 default。** 我先用 `import platform from '@/platform'` 报 "default is not exported",改成 `import { platform } from '@/platform'` 通过。

**问题 3:vue-i18n common 缺 `confirm` 字段。** openConfirm 用了 `t('common.confirm')` 作 fallback,但 common 里没这个 key。在 zh/en 各自补一行。
