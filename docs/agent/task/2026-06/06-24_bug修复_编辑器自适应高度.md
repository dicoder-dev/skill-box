## 4. 问题与方案

**问题:.md-editor 高度固定 320px,下方留白。**
根因:.detail-pane 是 `display: flex; flex-direction: column; overflow-y: auto`,但 .detail-body 是普通块级,非 flex item with `flex:1`,所以不会自然撑到 .detail-pane 底部。

方案:
- `.detail-body` 改为 `flex: 1; display: flex; flex-direction: column; min-height: 0` — 占满 .detail-pane 剩余高度,并自己变成 flex 容器
- `.md-editor` 加 `flex: 1` — 在 .detail-body 内填满剩余高度
- 保留 `min-height: 320px` — 内容少 / viewport 小时不至于塌成一行
- `resize: vertical` → `resize: none` — 自适应高度模式下,用户拖拽会破坏布局
