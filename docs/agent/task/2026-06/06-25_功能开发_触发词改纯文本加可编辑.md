## 4. 问题与方案

### 触发词编辑态数据流
- 新增 `editTriggersText`(字符串,换行分隔)
- `startInlineEdit` 时把 `currentMeta.triggers` 数组 join('\n')
- `cancelInlineEdit` 时清空
- `saveInlineEdit` 时 split + trim + filter(Boolean) 还原数组,同时塞进 manifest.triggers + frontmatter + currentMeta(用户立刻能看到新触发词)

### rebuildSkillMd 改签名
- 旧:`rebuildSkillMd(newBody)`,triggers 固定从 currentMeta 读
- 新:`rebuildSkillMd(newBody, newTriggers)`,让 saveInlineEdit 显式传入新触发词
- 默认行为兼容:不传 newTriggers 时回退到 currentMeta.triggers

### 编辑触发词校验
- 不强制非空:用户可以清空触发词(允许,但 manifest.triggers 会是 [])
